# Портфолио моих проектов в области DataScience



## 1. Аналитические проекты



- #### Исследование спроса на авиабилеты во время музыкальных фестивалей

Каталог: ***Air-ticket-demand-analysis***

Задача:  Проанализировать, на каких самолетах и в какие города летают больше. Выяснить, отличается ли спрос на авиабилеты во время проведения музыкальных фестивалей.

Навыки: `pandas, numpy, collections, matplotlib, seaborn, plotly`, feature engineering, A/B тестирование, перестановочный тест

Особенности работы: малый объём данных (10 объектов для A/B тестирования), данные получены из базы данных при помощи SQL-запросов, feature engineering на основе дополнительного анализа особенностей отрасли, интерактивная визуализация plotly



- #### Прогноз успешности компьютерных игр

Каталог: ***Computer-games-succsess-analysis***

Задача:  выявить закономерности, определяющие успешность компьютерной игры, сделать прогноз продаж на следующий год

Навыки: `pandas, numpy, scipy, collections, matplotlib, seaborn `, feature engineering, A/B тестирование, перестановочный тест, t-тест, ANOVA (дисперсионный анализ), корреляционный анализ (метод Спирмена), проверка распределений на нормальность

Особенности работы: потребовалось глубокое погружение в тему; из большого объёма данных выделен период времени, актуальный для анализа; выделено *current gen* домашних консолей; изучено множество кросс-взаимосвязей, т.к. в различный регионах мира и на различных платформах популярны разные жанры; подробный графический анализ (barhplot, boxplot, heatmap, scatterplot, countplot, pie, hlines etc.); объёмный блок статистического анализа; построен портрет типичного пользователя каждого из регионов; объёмный блок выводов содержит исчерпывающую информацию и рекомендации в зависимости от региона, пар жанр-платформа, конкретных игр, учитывают оценки критиков, пользователей, поколения консолей и многое другое.



- #### Сравнительный анализ доходности тарифов сотовой связи

Каталог: ***Mobile-communication-tariffs-profit-analysis***

Задача: на небольшой выборке клиентов сравнить доходность двух тарифов сотовой связи

Навыки:  `pandas, numpy, scipy, statsmodels.api, matplotlib, seaborn`,  предобработка данных (поиск аномалий, очистка, заполнение пропусков, объединение датасетов), A/B тестирование, анализ , перестановочный тест, визуализация результатов

Особенности работы: объединение нескольких датасетов, feature engineering в условиях небольшой разбалансированной выборки (500 пользователей из разных городов, с различным временем пользования тарифом),  проверка данных на нормальность распределения, оценка размера эффекта, A/B тестирование (критерий Манна-Уитни и перестановочный тест)



- #### Исследование объявлений о продаже квартир в Санкт-Петербурге и Ленинградской области

Каталог: ***SPb-real-estate-cost-analysis***

Задача: Выяснить, от чего зависит рыночная стоимость недвижимости в Санкт-Петербурге и Ленинградской области 

Навыки:`pandas, numpy, matplotlib, seaborn`, предобработка данных (поиск аномалий, очистка, заполнение пропусков), исследовательский и корреляционный анализ, визуализация результатов.

Особенности работы: серьёзный блок по очистке данных, объёмный исследовательский блок с визуализацией в seaborn, корреляционный анализ и тепловые карты для отображения результатов.



## 2. Machine learning проекты



- #### Классификация клиентов банка по их лояльности

Каталог: ***Bank-customers-classificator***

Задача: построить модель бинарной классификации для прогноза, прекратит клиент пользоваться услугами банка или нет

Навыки: `pandas, numpy, collections, random, sklearn, matplotlib, seaborn `, предобработка данных (поиск аномалий, очистка, заполнение пропусков), исследовательский анализ, статистический анализ (перестановочный тест, A/B тестирование) визуализация результатов, машинное обучение.

Особенности работы:  дисбаланс классов в данных; объёмный исследовательский блок с визуализацией в seaborn; заполнение пропусков <code>IterativeImputer и KNNImputer</code>, сравнение методов; масштабирование признаков <code>StandardScaler</code>, балансировка классов (upsampling и downsampling); настройка порога отнесения к классу по `predict_proba`; 



- #### Определение тональности комментариев

Каталог: ***Binary-text-classification***

Задача:  Нужно обучить модель классифицировать комментарии на позитивные и негативные. Язык комментариев - английский.

Навыки:  `pandas, numpy, re, torch, csv, transformers (BERT), spacy, nltk, sklearn, catboost`, лемматизация, векторизация текстов, машинное обучение

Особенности работы: большой блок по поиску оптимальных моделей и методов векторизации текстов; обучен spaCy TextCategorizer; для получения  BERT embeddings набор данных оптимизирован по особому алгоритму ("smart batching") (удалось добиться ускорения процесса в несколько раз)



- #### Разработка модели для прогноза рыночной стоимости автомобилей с пробегом

Каталог: ***Cars-cost-forecast***

Задача: построить модель для определения рыночной стоимости автомобиля

Навыки:  `pandas, numpy, math, datetime, collections, ipywidgets, sklearn, catboost, lightgbm, xgboost, category_encoders, matplotlib, seaborn`, предобработка данных (поиск аномалий, очистка, заполнение пропусков), исследовательский анализ, визуализация результатов, машинное обучение.

Особенности работы: данные загрязнены аномалиями и содержат пропуски; собственный метод заполнения пропусков и кодирования признаков (написал специальные классы, поддерживающие fit и transform); свой pipeline (написал класс, поддерживающий методы transform и predict) и gridsearch; сравнение catboost, xgboost и lightgbm по значениям метрик и скорости работы; графический анализ; успешная проверка работы своего класса на сырых данных



- #### Определение возраста человека по его фотографии

Каталог: ***CV-age-recognision-by-face***

Задача: Построить модель, которая по фотографии определит приблизительный возраст человека

Библиотеки и навыки:  `pandas, numpy, PIL, matplotlib, seaborn, Keras` ,  исследовательский анализ, визуализация результатов, обучение свёрточных нейронных сетей.

Особенности работы: развёрнутый анализ набора данных, обучение собственной модели на gpu kaggle (MAE = 5.8542), обучение модели на базе ResNet на Яндекс.Практикум



- #### Прогноз эффективности восстановления золота из золотосодержащей руды

Каталог: ***Gold-recovery-forecast***

Задача: построить модель для прогноза коэффициента восстановления золота (recovery) из золотосодержащей руды

Навыки: <code>pandas, numpy, math, sklearn, catboost, xgboost, matplotlib, seaborn</code>, предобработка данных (поиск аномалий, очистка, заполнение пропусков), исследовательский анализ, визуализация результатов, машинное обучение.

Особенности работы: глубокое погружение в тему, построение подробной схемы обогащения руды (более 80 специальных параметров); входные данные с большим количеством пропусков и аномалий; сложный характер взаимосвязей признаков со значением целевого признака; объёмный исследовательский анализ с подробной визуализацией; два подхода к прогнозу: прямой прогноз recovery и расчёт recovery на базе прогноза доли золота в концентрате и хвостах; сравнение ансамбля моделей для каждого этапа процесса с моделью, описывающей все этапы; сравнение регрессионных моделей и моделей на основе бустинга на решающих деревьях



- #### Защита данных клиентов с сохранением качества работы моделей машинного обучения

Каталог: ***Information-security***

Задача: разработать метод шифрования данных клиентов страховой компании, не меняющий качество работы модели линейной регрессии

Навыки: <code>pandas, numpy, sklearn, matplotlib, seaborn</code>, предобработка данных, исследовательский анализ, feature engineering, матричное умножение, транспонирование матриц, вычисление детерминанта, проверка обратимости

Особенности работы: изучение методов шифрования с помощью аналитических преобразований; строгое математическое доказательство применимости выбранного метода; feature engineering с существенным улучшением целевой метрики




- #### Выбор региона для разработки нефтяных месторождений

Каталог: ***Oil-field-volume-forecast***

Задача: разработать модель для прогноза объёма нефти в месторождении. Оценить риск убытков при разработке региона на основании прогнозов модели. Рекомендовать один из регионов к разработке.

Навыки: `pandas, numpy, scipy, collections, matplotlib, seaborn, plotly, sklearn, hdbscan, catboost, xgboost`, feature engineering, визуализация, статистический анализ (bootstrap, перестановочный тест, A/B тестирование), машинное обучение.

Особенности работы: объёмный блок функций для статистического анализа (авторские функции для перестановочного теста и bootstrap), feature engineering на основании авторских методов кластеризации, интерактивная визуализация в plotly.



- #### Разработка модели для подбора тарифа сотовой связи

Каталог: ***Recommendation-system-for-mobile-phone-rates***

Задача: разработать модель, которая порекомендует для пользователя один из двух тарифов сотовой связи

Библиотеки и навыки: `pandas, numpy, scipy, collections, datetime, random, sklearn, matplotlib, seaborn`,  визуализация, статистический анализ, машинное обучение.

Особенности работы: наглядный графический анализ, корреляционный анализ, оценка влияния гиперпараметров на качество модели по lineplot, сравнение моделей по скорости работы, проверка адекватности, использование classification_report при оценке качества моделей



- #### Модель для прогноза количества заказов такси на следующий час

Каталог: ***Taxi-orders-forecast***

Задача: спрогнозировать количество заказов такси на следующий час (есть исторические данные о количестве заказов такси в разное время)

Библиотеки и навыки: `pandas, numpy, sklearn, statsmodels, catboost, xgboost, matplotlib, seaborn`,  визуализация, декомпозиция временного ряда, автокорреляционный анализ, машинное обучение.

Особенности работы: feature engineering на основе декомпозиции временного ряда `seasonal_decompose` и расчёта автокорреляций `plot_acf`, наглядный графический анализ, сравнение моделей градиентного бустинга и регрессии.
